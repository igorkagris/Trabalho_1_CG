O desenvolvimento foi realizado através do Visual Studio na linguagem Python 3.12.1 com a interface gráfica do Tkinter 8.6 em um ambiente com Windows 11 na versão 22h2:

Para a execução do código, faz-se necessária a instalação do python através do site oficial 'https://www.python.org/downloads/' .
Após a instalação do python, abrir o Prompt de comando do Windows (CMD) e executar o comando "pip install tk" para a instalação do tkinter.
Com tudo instalado, o proximo passo é executar o código através do CMD com o comando "python fill_poly.py".
